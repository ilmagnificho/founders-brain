# YC Startup School AI 챗봇 (FoundersBrain) - 기획서

## 📌 핵심 요약
- **소스:** YC Startup School 25개 영상 기반
- **기술:** Gemini Flash API 활용
- **비용:** 월 운영비 약 5만원 (초기)
- **목표:** 유료 전환 시 월 20~50만원 순수익 (사이드 프로젝트)

## 1. 서비스 범위 (Lean Scope)

### 1.1 콘텐츠 소스
| 항목 | 수량 | 예상 분량 | 토큰 추정 |
| :--- | :--- | :--- | :--- |
| YC Startup School | 25개 영상 | 약 12시간 | ~150,000 토큰 |
| 주요 주제 | 창업 기초 | PMF, 투자 등 | - |

**플레이리스트:** `youtube.com/playlist?list=PLQ-uHSnFig5M9fW16o2l35jrfdsxGknNB`

### 1.2 핵심 기능 (MVP)
1. **Q&A:** 한국어 질문 → 영어 콘텐츠 검색 → 한국어 답변
2. **출처 제공:** 답변의 근거가 되는 영상 링크 제공 (클릭 시 원본 YouTube로 이동)
3. **타임스탬프:** 해당 내용이 나오는 시점으로 바로 이동 (`t={seconds}`)
4. **사용 제한:** 일일 무료 질문 제한 (5회) + 유료 무제한

### 1.3 타겟 사용자
- YC 지원 준비 중인 한국 창업자
- 스타트업 창업을 고민하는 직장인
- 영어 콘텐츠 소화가 어려운 예비 창업자
- YJ의 IR 코칭 서비스 기존 고객 (교차 판매)

## 5. 기술 스택 및 개발 방법

### 5.1 YJ 맞춤 개발 스택
| 레이어 | 도구 | 비용 | 비고 |
| :--- | :--- | :--- | :--- |
| 개발 환경 | Antigravity | 무료~$20/월 | YJ 경험 있음 ✓ |
| 버전 관리 | GitHub | 무료 | 자동 푸시 ✓ |
| 호스팅 | Vercel | 무료 (Hobby) | 자동 배포 ✓ |
| LLM API | Gemini 3 Flash | 종량제 | 한국어 우수 |
| Vector DB | Supabase pgvector | 무료 티어 | PostgreSQL 기반 |
| 결제 | Toss Payments | 3.3% 수수료 | 한국 최적화 |
| 인증 | 카카오 로그인 | 무료 | 한국 사용자 친화 |

### 5.2 개발 워크플로우
1. **Claude Code/Antigravity** → GitHub (자동 푸시) → Vercel (자동 배포)
2. **트랜스크립트 수집:** `youtube-transcript-api`로 25개 영상 추출
3. **임베딩 생성:** Gemini text-embedding으로 벡터화 → Supabase 저장
4. **RAG 파이프라인:** 질문 → 유사 청크 검색 → Gemini로 답변 생성
5. **프론트엔드:** Next.js + Tailwind (Vercel 최적화)

### 5.3 타임스탬프 기능 구현
- YouTube 트랜스크립트 API는 각 문장의 시작 시간을 제공
- 트랜스크립트 저장 시 타임스탬프도 함께 저장
- 답변 생성 시 출처 청크의 타임스탬프 추출
- 링크 형식: `youtube.com/watch?v={video_id}&t={seconds}`
